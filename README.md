## How to Run Locally

Since this project uses the `fetch()` API to load the `quotes.json` file, it is best viewed through a local server.

1.  **Clone the Repository:**
    ```bash
    git clone [https://github.com/your-username/inspire-me.git](https://github.com/your-username/inspire-me.git)
    ```
2.  **Open with Live Server:**
    * If you use **VS Code**, install the "Live Server" extension.
    * Right-click `index.html` and select **"Open with Live Server"**.
3.  **Manual Way:**
    * Simply open the folder and drag `index.html` into your browser (Note: Some browsers may block JSON loading for security; use a local server for the best experience).
